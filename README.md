# Podcast del Doctor

## Sobre el Podcast

Podcast sobre programació i tecnologia per David Rodenas. Contingut generat amb intel·ligència artificial per fomentar l'esperit crític i el debat tècnic.

## Disclaimer Important

**El contingut d'aquest podcast està generat amb intel·ligència artificial.** Encara que està basat en fonts reals, pot contenir interpretacions o matisos que no encaixin completament amb la realitat.

**Sempre es proporcionen les fonts originals** perquè qui ho desitgi pugui consultar la informació directament.

## Estructura del Repositori

```
/
├── episodes/           # Fitxers MP3 dels episodis
├── _episodes/          # Fitxers markdown amb metadades dels episodis
├── sources/            # Fonts i transcripcions
├── scripts/            # Scripts d'automatització
├── assets/             # Imatges, logo, etc.
├── _layouts/           # Plantilles Jekyll
├── _config.yml         # Configuració Jekyll
└── feed.xml            # RSS feed del podcast
```
